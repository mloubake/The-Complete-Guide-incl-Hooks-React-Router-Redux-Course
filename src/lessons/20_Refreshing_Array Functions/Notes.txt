Some array functions: filter, sort, map and so on.

const numbers = [1, 2, 3]

This function will be executed on each element.

const doubleNumbers = numbers.map((number) => {
    return number * 2;
})

It will return a new brand array with the new values

console.log(numbers) //Result: [1, 2, 3]
console.log(doubleNumbers) //Result: [2, 4, 6]
